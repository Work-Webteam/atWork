INTRODUCTION
------------

Add a Like and Dislike field on every entity (like content, comments etc)
Get instant statistics on content itself.

REQUIREMENTS
------------

This module does not have any dependency.

INSTALLATION
------------

* Install as any other drupal module.

CONFIGURATION
-------------

* Add the Like/Dislike field to any entity type
* A user can either like/dislike the piece of content.
